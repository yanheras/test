Extension: Condition Category Remark Extension

Official URL: http://hl7.org/fhir/us/davinci-ra/StructureDefinition/ra-ccRemark Version: 3.0.0-ballot
Standards status: Trial-use Active as of 2026-03-08 Maturity Level: 2 Computable Name: ConditionCategoryRemark
Other Identifiers: OID:2.16.840.1.113883.4.642.40.29.42.1

Copyright/Legal: Used by permission of HL7 International, all rights reserved Creative Commons License

The Condition Category Remark extension allows a coded and/or narrative remark be added to a Condition Category (CC) coding gap that is on a Payer generated risk adjustment coding gap report by a provider, an organization, or a software.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: MeasureReport.group

Usage info

Usages:

  • Use this Extension: Risk Adjustment Coding Gap MeasureReport
  • Examples for this Extension: MeasureReport/ra-measurereport01-with-remark

You can also check for usages in the FHIR IG Statistics
