Extension: Group Reference Extension

Official URL: http://hl7.org/fhir/us/davinci-ra/StructureDefinition/ra-groupReference Version: 3.0.0-ballot
Standards status: Trial-use Active as of 2026-03-08 Maturity Level: 2 Computable Name: GroupReference
Other Identifiers: OID:2.16.840.1.113883.4.642.40.29.42.9

Copyright/Legal: Used by permission of HL7 International, all rights reserved Creative Commons License

This extension identifies which Condition Category (CC) the supporting evidence (represented by MeasureReport.evaluatedResource) is for.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: MeasureReport.evaluatedResource
    • Element ID: DetectedIssue

Usage info

Usages:

  • Use this Extension: Risk Adjustment Data Exchange MeasureReport and Risk Adjustment Coding Gap MeasureReport
  • Examples for this Extension: Bundle/ra-bundle01, MeasureReport/ra-measurereport01-with-remark, MeasureReport/ra-measurereport01, MeasureReport/ra-measurereport02... Show 4 more, MeasureReport/ra-measurereport03, MeasureReport/ra-measurereport04, MeasureReport/ra-measurereport05 and MeasureReport/ra-measurereport06
